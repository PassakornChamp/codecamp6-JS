Grid Flex ความแตกต่าง
ใช้จัดตำแหน่งแบบสองมิติ ทั้งแบบตั้งและนอน เช่นใช้ช้ในกาจัดโครงหน้าเพจบนล่างและข้าง ใช้จัดตำแหน่งในรูปแบบแบบมิติเดียว เช่นใช้จัดตำแหน่งหนึ่งส่วนของหน้าเพจ ใช้งานคนละจุดประสงค์กันถึงแม้ว่าจะสามารถทำงานคล้ายๆกัน

Justify Flex ความแตกต่าง
คือการจัดตำแหน่งแนวนอน คือการจัดตำแหน่งแนวตั้ง ใช้งานคนละจุดประสงค์กัน
